# customer_shopping_behavior
## Project Overview
This project is a complete end-to-end data analysis project based on retail customer data. The main aim is to understand customer behavior and turn raw data into useful insights.

In this project, I worked on different stages of data analysis similar to what a data analyst does in a real company.

🔍 What I Did in This Project
🐍 Data Cleaning and Exploration (Python)

First, I loaded the dataset in Python and explored it to understand the structure.
I handled missing values, cleaned the data, and prepared it for further analysis.

## 🗄️ Data Analysis (SQL)

After cleaning the data, I loaded it into a SQL database.
I wrote SQL queries to answer business questions like:

- Which customers spend the most?

- What are the most popular products?

-How customer behavior changes over time
## 📊 Dashboard Creation (Power BI)

I connected the SQL database to Power BI and created an interactive dashboard.
The dashboard shows important insights like sales trends, customer segments, and overall performance.

## 📝 Report and Insights

Finally, I summarized all the findings in a report.
I explained the key insights and gave some basic recommendations based on the data.

## 🎯 Purpose of the Project

The goal of this project is to practice and demonstrate:

- Data cleaning and analysis using Python
- Writing SQL queries for real-world problems
- Creating dashboards using Power BI
- Turning data into meaningful insights

## 🛠️ Tools Used
- Python
- SQL
- Power BI
